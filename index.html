What is this ?

A simple method for aproximating π using some random points.

How does it work ?

I stumbled on this method while surfing youtube (source).

What we have here is a circle inscribed in a square. The area of the square is \(A_{sqr} = Length^2 = 4R^2\); While the area of the circle is \(A_{circ} = {\pi}R^2\). If we divide \(\frac{A_{circ}}{A_{sqr}} = \frac{\pi}{4}\) Or \( \pi = 4\frac{A_{circ}}{A_{sqr}} \). The next thing to understand is that when we keep generating random points we start filling the Area !!!

It generates random points (x, y) and counts them (Ntotal = total nr. of points generated). Each point is checked it hits the circle (Ncirc = points inside the circle).

All it's left to do is to calculate \(\pi = 4\frac{A_{circ}}{A_{sqr}} = 4\frac{N{circ}}{N{total}}\)
